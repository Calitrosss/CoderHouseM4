# 2da. Pre-entrega Proyecto Final

---

- Para instalar las dependencias: `npm i`
- Para iniciar: `nodemon main`
- Agregar archivo .env en el directorio raíz con los siguientes parámetros:
  - PORT=
  - MONGO_URL=
  - SECRET_KEY=
  - ADMIN_EMAIL=
  - ADMIN_PASSWORD=
  - GITHUB_CLIENT_ID=
  - GITHUB_CLIENT_SECRET=
  - GITHUB_CALLBACK_URL=
